# balingo-test
Prototype project for a new Balatro Bingo gameplay format

## V1 Feature Requirements:

- Bingo Board generator
- Center square is always a "Beat this Seed:" with the seed being a random Wordle word
- Controls and visuals for objectives to be paired with stakes indicators
- Timer for when a player reaches a win condition
- Timer for objectives not being completed for a certain time (triggering a game end)
